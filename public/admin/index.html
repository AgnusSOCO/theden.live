available embeds

Use these HTML comments in your markdown to add rich components:

<!-- chart:BTCUSD --> TradingView chart (symbol, optional interval like :weekly)
<!-- price:bitcoin --> Live crypto price with 24h change
<!-- trade:BTC|long|42000|45000|0.5 --> Trade card (symbol|direction|entry|exit|size)
<!-- pnl:+15.5 --> Profit/loss badge
<!-- coin:ethereum --> Crypto logo with symbol
<!-- callout:tip:Your message --> Callout box (info/warning/success/tip/error)
<!-- balance:1047.50:1000 --> Account balance display (current:starting)
